# SQL-Employee-Data-Analysis
This project demonstrates SQL queries used for analyzing employee data.
## Tools Used:
-- SQL Server
-- SQL

# Concepts Covered
-- Table Creation
-- Primary key and Foreign key
-- Subqueries
-- Group By and  Having
-- Window Functions (Row_Number,Dense_Rank)
-- Common Table Expression (CTE)
-- Duplicate Record Handling
-- Ranking Queries
## Business Problems Solved
1. Find second highest salary.
2. Find third highest salary
3. Identify employees earning more than their manager
4. Find duplicate employee records
5. Calculate department-wise average salary
6. Identify top 3 highest paid employees.
7. find employees hired in the last 6 months.
8. find employees whose name starts and ends with vowel.
9. delete duplicate records but keep one copy.
10. find departments having more than 5 employees.
